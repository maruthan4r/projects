Tk()

Creates the root widget(Main Window).
If you don't initialize it you call it with .master
Example:
from tkinter import *
frame = Frame(width=100,height=100) #creates a frame frame.grid() #Places the frame in the grid
frame.master.title("Window title") #sets title of the window mainloop() #runs the main loop
